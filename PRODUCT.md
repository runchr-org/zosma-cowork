# Product

## Register

product

## Users

Developers and AI power users who run agentic tasks on their desktop. They are in a focused work session: writing code, processing files, directing multi-step agent runs. They stay in the app for stretches of 20–90 minutes, switching between sessions, watching tool calls play out, and nudging the agent mid-run. Ambient light varies; the interface is used equally in dim home offices and bright co-working spaces.

## Product Purpose

Zosma Cowork is a desktop AI coworker: a native Tauri shell around the pi-mono agent SDK that gives every pi extension, skill, and prompt template a first-class UI. It handles streaming chat, thinking display, tool-call timelines, multi-turn sessions, and model selection so users can direct agentic work without touching a terminal. Success looks like: the agent disappears into the task and the user stops noticing the shell.

## Brand Personality

Precise, calm, open-source honest. Three words: focused, grounded, capable.

The brand does not perform intelligence. It gets out of the way. Delight is earned through speed and clarity, not decoration.

## Anti-references

- ChatGPT web: too consumer, too rounded, too many gradients
- GitHub Copilot Chat: correct register but too corporate-neutral; no personality
- "AI-maximalist" dark dashboards with glowing purple/cyan grids and animated particles
- Cream/sand/beige warm-tinted backgrounds (the 2026 AI default)
- Any interface where the model branding overshadows the user's work

## Design Principles

1. **The agent disappears into the task.** Chrome, chrome, chrome only when it earns its keep. If the user is staring at the shell instead of the output, something is wrong.
2. **Calm confidence over anxious busyness.** Motion communicates state, not effort. Loading states are quiet; completion states are instant.
3. **Open-source honest.** No dark patterns, no upsell surfaces, no engagement hooks. The interface trusts the user to know what they need.
4. **Density earns its place.** Session lists, tool timelines, and model selectors can be dense because users are fluent. Onboarding surfaces breathe.
5. **Green is the only accent.** The Oliver AI green (#98F46F) is the single signal color: active states, primary actions, live indicators. Nothing else competes.

## Accessibility & Inclusion

WCAG 2.1 AA minimum. Reduced-motion preference respected on all transitions. Keyboard-navigable session list and model selector. Sufficient contrast on both light and dark themes (4.5:1 for body, 3:1 for large/bold).
